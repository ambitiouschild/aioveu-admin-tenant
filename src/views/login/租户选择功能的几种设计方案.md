当前登录页面没有租户选择功能。这是一个常见问题，因为多租户系统需要让用户选择要登录的具体租户。

以下是**租户选择功能的几种设计方案**，从最简单到最复杂：

------

## 方案一：下拉框选择（最简单直观）

在用户名输入框后添加一个租户选择下拉框。

```
<template>
  <div class="auth-panel-form">
    <h3 class="auth-panel-form__title" text-center>{{ t("login.login") }}</h3>
    <el-form
      ref="loginFormRef"
      :model="loginFormData"
      :rules="loginRules"
      size="large"
      :validate-on-rule-change="false"
    >
      <!-- 用户名 -->
      <el-form-item prop="username">
        <el-input v-model.trim="loginFormData.username" :placeholder="t('login.username')">
          <template #prefix>
            <el-icon><User /></el-icon>
          </template>
        </el-input>
      </el-form-item>

      <!-- 租户选择（新增） -->
      <el-form-item prop="tenantId">
        <el-select
          v-model="loginFormData.tenantId"
          :placeholder="t('login.selectTenant')"
          filterable
          clearable
          class="w-full"
          :loading="tenantLoading"
          @focus="loadTenants"
        >
          <template #prefix>
            <el-icon><OfficeBuilding /></el-icon>
          </template>
          <el-option
            v-for="tenant in tenantList"
            :key="tenant.id"
            :label="tenant.name"
            :value="tenant.id"
          >
            <div flex items-center justify-between>
              <span>{{ tenant.name }}</span>
              <el-tag v-if="tenant.code" size="small">{{ tenant.code }}</el-tag>
            </div>
          </el-option>
        </el-select>
      </el-form-item>

      <!-- 密码 -->
      <!-- ... 其余代码保持不变 ... -->
    </el-form>
  </div>
</template>

<script setup lang="ts">
import { ref, onMounted } from 'vue';
import { OfficeBuilding } from '@element-plus/icons-vue';
import type { TenantItem } from '@/types/api';

// 租户相关
const tenantList = ref<TenantItem[]>([]);
const tenantLoading = ref(false);

// 登录表单数据
const loginFormData = ref({
  username: 'admin',
  password: '123456',
  tenantId: '', // 新增租户ID字段
  captchaId: '',
  captchaCode: '',
  rememberMe: false,
});

// 加载租户列表
async function loadTenants() {
  if (tenantList.value.length > 0) return; // 已加载则不重复加载
  
  tenantLoading.value = true;
  try {
    // 调用API获取租户列表
    const response = await AuthAPI.getTenantList(loginFormData.value.username);
    tenantList.value = response.data || [];
    
    // 如果只有一个租户，默认选中
    if (tenantList.value.length === 1) {
      loginFormData.value.tenantId = tenantList.value[0].id;
    }
  } catch (error) {
    console.error('加载租户列表失败:', error);
  } finally {
    tenantLoading.value = false;
  }
}

// 监听用户名变化，自动加载租户
watch(() => loginFormData.value.username, (newUsername) => {
  if (newUsername && newUsername.length >= 3) {
    // 可以添加防抖
    tenantList.value = []; // 清空之前的租户列表
    loginFormData.value.tenantId = ''; // 清空选择的租户
  }
}, { immediate: true });

// 表单验证规则更新
const loginRules = computed(() => ({
  username: [
    { required: true, trigger: 'blur', message: t('login.message.username.required') },
  ],
  tenantId: [
    { required: true, trigger: 'change', message: t('login.message.tenant.required') },
  ],
  // ... 其他规则保持不变
}));
</script>
```

## 方案二：分步登录（更复杂的场景）

如果租户数量多或有复杂的租户层级，可以采用分步登录：

```
<template>
  <!-- 第一步：输入用户名 -->
  <div v-if="step === 1" class="auth-panel-form">
    <h3>选择账户</h3>
    <el-input
      v-model="username"
      placeholder="请输入用户名"
      @keyup.enter="loadUserTenants"
    />
    <el-button @click="loadUserTenants">下一步</el-button>
  </div>

  <!-- 第二步：选择租户 -->
  <div v-else-if="step === 2" class="auth-panel-form">
    <h3>选择租户</h3>
    <div class="tenant-grid">
      <div
        v-for="tenant in tenantList"
        :key="tenant.id"
        class="tenant-card"
        @click="selectTenant(tenant)"
      >
        <div class="tenant-avatar">{{ tenant.name.charAt(0) }}</div>
        <div class="tenant-name">{{ tenant.name }}</div>
        <div class="tenant-code">{{ tenant.code }}</div>
      </div>
    </div>
    <el-button @click="step = 1">返回</el-button>
  </div>

  <!-- 第三步：输入密码 -->
  <div v-else class="auth-panel-form">
    <div class="login-header">
      <div class="user-tenant-info">
        <span>{{ username }} · {{ selectedTenant?.name }}</span>
        <el-link @click="step = 2">切换租户</el-link>
      </div>
    </div>
    <!-- 原来的密码和验证码输入部分 -->
  </div>
</template>
```

## 推荐方案

对于大多数系统，**方案一（下拉框选择）** 是最佳选择，因为：

1. **用户体验好**：一目了然，操作简单
2. **开发简单**：只需添加一个选择框
3. **兼容性强**：适合租户数量适中的场景
4. **易于扩展**：可以添加搜索、分组等功能

**实现建议**：

1. 在用户输入用户名后，自动查询该用户关联的租户
2. 租户列表可以本地缓存，避免重复请求
3. 添加"记住租户"选项，下次自动选择
4. 对于单租户用户，可以自动选择并隐藏选择框
